# greenly-earth

Greenly — SMB-focused carbon accounting platform (Paris / London / New York). Automates corporate carbon footprint across Scope 1/2/3, produces Bilan Carbone, GHG Protocol, CSRD, CBAM, EUDR, SBTi, TCFD/IFRS, EcoVadis, and California climate disclosure deliverables, and exposes a Carbon Analytics REST API for transaction-level emissions enrichment.

## API

- **Greenly Carbon Analytics API (v1.5)** — REST service that enriches bank-card / expense transactions with carbon footprint analytics, classifies purchase categories, surfaces lower-carbon alternatives, and books verified carbon-offset volumes.
  - Production: `https://api.greenly.earth/v1.5`
  - Sandbox: `https://apisandbox.greenly.earth/v1.5`
  - Auth: `api-key` header (request via contact@greenly.earth)
  - Spec: [openapi/greenly-carbon-analytics-api-openapi.yml](openapi/greenly-carbon-analytics-api-openapi.yml)
